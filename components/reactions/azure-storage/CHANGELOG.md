# Changelog

## [0.1.0] - 2026-03-06

### Added

- Initial Azure Storage reaction crate with Blob, Queue, and Table targets
- Template-based data mapping using Handlebars
- Descriptor/OpenAPI schema support for dynamic plugin loading
- Azurite-based integration tests for INSERT/UPDATE/DELETE flows
